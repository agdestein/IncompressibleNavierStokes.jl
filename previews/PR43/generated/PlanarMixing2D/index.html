Planar mixing - 2D

Planar mixing example, as presented in [3].

We start by loading packages. A Makie plotting backend is needed for plotting. GLMakie creates an interactive window (useful for real-time plotting), but does not work when building this example on GitHub. CairoMakie makes high-quality static vector-graphics plots.

using CairoMakie
using IncompressibleNavierStokes

Output directory

output = "output/PlanarMixing2D"
"output/PlanarMixing2D"

Viscosity model

Re = 500.0
500.0

Boundary conditions: Unsteady BC requires time derivatives

ΔU = 1.0
Ubar = 1.0
ϵ = (0.082Ubar, 0.012Ubar)
n = (0.4π, 0.3π)
ω = (0.22, 0.11)
U(dim, x, y, t) =
    dim() == 1 ?
    1.0 + ΔU / 2 * tanh(2y) + sum(@. ϵ * (1 - tanh(y / 2)^2) * cos(n * y) * sin(ω * t)) :
    0.0
dUdt(dim, x, y, t) =
    dim() == 1 ? sum(@. ϵ * (1 - tanh(y / 2)^2) * cos(n * y) * ω * cos(ω * t)) : 0.0
boundary_conditions = (
    # x left, x right
    (DirichletBC(U, dUdt), PressureBC()),

    # y rear, y front
    (PressureBC(), PressureBC()),
)
((DirichletBC{typeof(Main.U), typeof(Main.dUdt)}(Main.U, Main.dUdt), PressureBC()), (PressureBC(), PressureBC()))

setup = Setup(x, y; Re, boundary_conditions);
